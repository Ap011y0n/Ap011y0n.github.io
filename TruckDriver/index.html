Truck driver simulator


Description

Truck driver simulator is a 3d physics playground, created using openGl and bullet3D library.
The main goal of the project was to implement a 3d physics library in a project, and play with it's features. the game consists on a handmade truck relying entirely on physics and constraints, where you must pick some balls with a grapple hook and store them in the city centre.

My Contribution


I was mainly tasked with creating the functions that allowed to create constraints with Bullet3D library, and then to build the truck relying on them. I also configurated the project to work with multiple cameras, so it was easier to use the hook. The OpenGl implementation was cooperative with the other team members.
